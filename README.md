# 🤖 AI Agent Capstone Advisor & Developer Sandbox

An immersive, production-grade learning sandbox and architectural planner designed to guide developers from Project Constitution to Automated Verification. This application acts as a friendly, highly interactive peer programmer and expert mentor (inspired by ChatGPT, Claude, and Gemini) to help students design, specify, and write secure, multi-agent workflows.

---

## ✨ Features & Capabilities

### ⚡ 21st-Century Premium Developer Console

A dark-mode desktop-optimized cockpit styled with razor-sharp telemetry, responsive transitions, high-contrast micro-animations, and fluid grid layouts.

### 💬 Supportive Cohort AI Advisor

An interactive chat companion that feels like a developer friend. It provides step-by-step guidance, code blocks, modular examples, and prompt suggestions to clear up roadblocks.

### 🛡️ Verification & Prerequisites Hub

A dedicated terminal-style hardware status tracker that syncs environment milestones, local packages, and sandboxing capabilities using modern container states.

### ⛓️ Multi-Agent Orchestration Visualizer

Direct visual rendering of parallel and sequential agent task-handoff workflows using standard Agent Development Kit (ADK) traces.

### ⚙️ CLI Code Simulator

Sandbox environment designed to run simulated Gherkin behavior-driven assertions (Given/When/Then) and mock sandbox test cases.

## 🛠️ The Tech Stack

### Frontend Core

* React 18 & TypeScript: For ultra-safe dynamic type compilation, stateful rendering, and modular architecture.
* Vite: Providing responsive bundle-building and high-speed cold starts.
* Tailwind CSS: Custom utility classes powering a specialized, dark space-slate layout accented with indigo, purple, and emerald visual glows.
* Motion: Fluid animations for conversational entries, hover feedbacks, and container changes.
* Lucide React: Clean vector icon structures.

### Backend Infrastructure

* Express & Node.js: Robust backend routing layer configured behind secure server-side proxy handlers.
* Google GenAI SDK (`@google/genai`): Interfacing with Gemini 3.5 Flash for lightning-fast, structured cognitive summaries and technical tutoring.
* Local Storage Persistence: Safely preserving user chat histories and tracking scores across restarts.

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js and npm installed.

### Installation

#### Clone your repository

```bash
git clone https://github.com/your-username/ai-agent-capstone-sandbox.git
cd ai-agent-capstone-sandbox
```

#### Install dependencies

```bash
npm install
```

#### Declare your Gemini API Key in a `.env` file at the root

```env
GEMINI_API_KEY=your_actual_api_key_here
```

#### Run the local development server

```bash
npm run dev
```

Open http://localhost:3000 in your web browser.

---

## 🏗️ Building for Production

To bundle and compile both the client assets and the Node.js Express server:

```bash
npm run build
npm start
```
